>Вот это не очено хорошо. В Линуксе есть Основная группа Пользователя, а есть дополнительные. В данном случае вы делаете группу sudo основной группой пользователя. GID основной группы используется для организации прав доступа. Группа sudo же не является основной ни для какого пользователя и ее GID не используется для управления правами на файлы. Короче говоря, нужно использовать параметр '-G' вместо '-g'.
